# 产品概述

## Arch-Graph：BIM 图数据库智能助手

下一代建筑信息模型（BIM）平台，将复杂的 3D 建筑模型转换为可交互的对话式知识图谱系统。

### 核心价值主张

- **自然语言 BIM 交互**：使用对话式命令查询和操作建筑模型，无需复杂的 CAD 界面
- **图数据库架构**：在 Neo4j 中存储空间关系、设备系统和建筑语义，实现智能查询
- **轻量化 3D 可视化**：通过 Speckle/Three.js 集成实现实时模型查看
- **AI 驱动的命令处理**：Gemini LLM 解析用户意图并生成结构化的 BIM 操作和 Cypher 查询
- **三向联动可视化**：3D 模型、图谱可视化、对话界面实时同步交互

### 目标用户

- 建筑师和施工专业人员
- 建筑设施管理人员
- 需要简化 BIM 访问的跨职能团队
- 需要建筑信息的非技术利益相关者

### 核心差异化优势（专利亮点）

- **BIM 模型语义 → 图数据库空间语言映射算法**：将 Revit/IFC 模型自动转换为 Neo4j 知识图谱
- **多模态界面**：3D 查看器 + 图谱可视化 + 对话式 UI 三向联动
- **空间关系分析**：房间连通性、管道穿越、设备系统关系的智能查询
- **自然语言 → Cypher 自动生成**：LLM 驱动的图查询生成
- **三分屏智能协作界面**：左侧上下分屏（模型+图谱）+ 右侧固定对话

### MVP 核心功能

1. **轻量化 BIM 模型查看器**：支持旋转/缩放/高亮/隐藏，点击元素查询语义
2. **Neo4j 图数据库**：存储空间、构件、系统、管线及其关系
3. **LLM 对话交互**：自然语言转 Cypher 查询和 Three.js 操作
4. **三分屏 UI**：左上（3D 模型）+ 左下（图谱）+ 右侧（对话），支持最大化/最小化

### 关键交互场景

- **空间查询**：分析每层空间分布，三向联动展示
- **路径导航**：查询空间路径，3D 动画播放
- **管线穿越分析**：分析管线穿越的空间，高亮显示关系链
